Loan Default Prediction

Advanced machine learning to assess loan default risk

Project Overview

This application uses a Logistic Regression model trained on financial data to predict the probability of loan default based on various financial and demographic factors.

The model was trained on a dataset containing information about customers and their loans, with the target variable being CreditScore which indicates default risk.

Key Features:
  • Predicts likelihood of loan default
  • Identifies key risk factors
  • Provides actionable insights
  • Visualizes model performance
Model Details:
  • Algorithm: Logistic Regression
  • Training data: 40,000+ loan records
  • Features: 20+ financial indicators
  • Preprocessing: Standard scaling & encoding
